Firera

Firera is a JavaScript library/framework that implements functional reactive programming(FRP) approach for building UI and other front-end tasks.
You can:
  • Explore a tutorial and information on Firera
  • Install/download latest stable release
  • See a full reference
  • Visit Firera repo on GitHub
  • Ask a question or join discussion on Gitter channel
There are some tools that can be used independently, but more often together with Firera.
These are Ozenfant - HTMl template engine, and Sche - a tool for creating declarative scenarios in browser.
