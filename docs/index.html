Firera is a library/framework that implements functional reactive programming(FRP) approach for building UI, communicating with backend etc.
You can:
  • Read more about Firera
  • Explore a brief Firera tutorial
  • See a full reference
  • Visit Firera repo on GitHub
  • Ask a question or discuss on Gitter channel
There some tools that are the members of Firera family. They can be used independently, but more often together with Firera.
These are Ozenfant - HTMl template engine, and Sche - a tool for creating declarative scenarios in browser.
